### Alert banner decision tree
- Source: `useStatePoll().data.breakers`
- Trigger: ANY of `halted`, `retryCount > 0`, `recentPhantomEscalations > 0`
- Copy (founder-speak, default):
  - halted: "Something paused progress. Click to see what's blocking."
  - retryCount>0: "A task retried ({n}). Usually self-heals; click for details."
  - phantomEscalations>0: "A background gate was escalated ({n}). Click to review."
- CTAs:
  - "Show details" → existing incident/queue route or inline expand (respect D-06: prefer nav to existing surface over new modal)
  - "Dismiss" → localStorage key `p13-alert-dismissed:{fingerprint}` where fingerprint = hash of trigger reason; banner reappears if a NEW trigger fires
- State cleared = NO banner (default)

### Ambient clock isolation
To avoid top-nav re-rendering every second (causing all nav-bar memoization busts):
```tsx
// components/shell/ambient-clock.tsx
"use client";
import { useEffect, useState } from "react";
export function AmbientClock() {
  const [time, setTime] = useState<string>(() => fmt(new Date()));
  useEffect(() => {
    const id = setInterval(() => setTime(fmt(new Date())), 1000);
    return () => clearInterval(id);
  }, []);
  return <span className="font-mono text-[10px] text-[color:var(--text-muted)] tabular-nums">{time}</span>;
}
function fmt(d: Date): string {
  return `${String(d.getHours()).padStart(2,"0")}:${String(d.getMinutes()).padStart(2,"0")}:${String(d.getSeconds()).padStart(2,"0")}`;
}
```
This component is local-state-only; setInterval inside its own scope, no context writes.

### agentVerbs contract
```typescript
// lib/copy/labels.ts
export type AgentVerbSet = "start_stop_archive" | "wake_spawn_hide";

const verbSets: Record<AgentVerbSet, { primary: string; stop: string; archive: string }> = {
  start_stop_archive: { primary: "Start", stop: "Stop", archive: "Archive" },
  wake_spawn_hide:    { primary: "Wake",  stop: "Hide", archive: "Hide"    },
};

export function agentVerbs(set: AgentVerbSet = "start_stop_archive") {
  return verbSets[set];
}

export function getAgentVerbSet(): AgentVerbSet {
  if (typeof window === "undefined") return "start_stop_archive";
  const v = localStorage.getItem("p13-agent-verbs");
  return v === "wake_spawn_hide" ? "wake_spawn_hide" : "start_stop_archive";
}
```
Agent card: read via `useSyncExternalStore` or a simple useState + localStorage sync on mount.
