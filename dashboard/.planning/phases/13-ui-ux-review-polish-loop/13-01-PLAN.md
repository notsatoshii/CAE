  <name>Task 3: Implement capture.sh — the reusable screenshot driver</name>
  <files>.planning/phases/13-ui-ux-review-polish-loop/scripts/capture.sh</files>
  <behavior>
    - Given `--phase before|after` and optional `--route <slug>`, write PNGs under `shots/{phase}/{viewport}-{mode}/{route-slug}.png`
    - Every protected route passes the `authjs.session-token` cookie from `storage-state.json`
    - Sets `explainMode` + `devMode` localStorage before navigating
    - Uses routes.json as the single source of truth for URLs, viewports, waits
    - Emits a manifest `shots/{phase}/MANIFEST.tsv` with columns: slug \t viewport \t mode \t url \t png_path \t byte_size
    - Exit non-zero if ANY screenshot capture fails (don't silently skip)
  </behavior>
  <action>
  `screenshot-url` at /usr/local/bin/screenshot-url is a Playwright Python wrapper — inspect it with `head -80 /usr/local/bin/screenshot-url` to see exact flags supported before writing the driver.

  Key discovery from inspection: `screenshot-url` takes `--viewport WxH`, `--wait-ms`, `--wait-selector`, `--theme`, but does NOT natively support cookies or localStorage injection. **Therefore capture.sh cannot just shell out to `screenshot-url`** — it needs its own Python Playwright wrapper that handles auth + localStorage + deterministic CSS injection.

  Write `capture.sh` as a bash dispatcher that invokes an inline Python Playwright script:

  ```bash
  #!/usr/bin/env bash
  set -euo pipefail
  PHASE_ARG="${1:-}"; shift || true
  if [[ "$PHASE_ARG" != "before" && "$PHASE_ARG" != "after" ]]; then
    echo "usage: capture.sh before|after [--route slug] [--viewport mobile|laptop|wide] [--mode founder|dev]" >&2
    exit 2
  fi
  ROUTE_FILTER=""; VIEWPORT_FILTER=""; MODE_FILTER=""
  while [[ $# -gt 0 ]]; do
    case "$1" in
      --route)    ROUTE_FILTER="$2"; shift 2 ;;
      --viewport) VIEWPORT_FILTER="$2"; shift 2 ;;
      --mode)     MODE_FILTER="$2"; shift 2 ;;
      *) echo "unknown arg: $1" >&2; exit 2 ;;
    esac
  done
  cd "$(dirname "$0")"
  export PLAYWRIGHT_BROWSERS_PATH="/usr/local/share/playwright-browsers"
  export PHASE="$PHASE_ARG" ROUTE_FILTER VIEWPORT_FILTER MODE_FILTER
  python3 - <<'PY'
  import json, os, sys, pathlib, time
  from playwright.sync_api import sync_playwright

  ROOT = pathlib.Path(__file__).parent if "__file__" in globals() else pathlib.Path.cwd()
  ROUTES = json.loads(pathlib.Path("routes.json").read_text())
  PHASE = os.environ["PHASE"]
  ROUTE_FILTER = os.environ.get("ROUTE_FILTER","")
  VIEWPORT_FILTER = os.environ.get("VIEWPORT_FILTER","")
  MODE_FILTER = os.environ.get("MODE_FILTER","")
  SHOTS = pathlib.Path(f"../shots/{PHASE}")
  SHOTS.mkdir(parents=True, exist_ok=True)
  MANIFEST = SHOTS / "MANIFEST.tsv"
  DETERMINISTIC_CSS = "* { animation: none !important; transition: none !important; caret-color: transparent !important; } *::before, *::after { animation: none !important; transition: none !important; }"
  fails = 0; shots = 0
  with sync_playwright() as pw:
      browser = pw.chromium.launch(headless=True)
      with MANIFEST.open("w") as mf:
          mf.write("slug\tviewport\tmode\turl\tpng_path\tbytes\n")
          for mode_name, matrix_viewports in ROUTES["mode_viewport_matrix"].items():
              if MODE_FILTER and mode_name != MODE_FILTER: continue
              mode_cfg = ROUTES["modes"][mode_name]
              for vp_name in matrix_viewports:
                  if VIEWPORT_FILTER and vp_name != VIEWPORT_FILTER: continue
                  w,h = map(int, ROUTES["viewports"][vp_name].split("x"))
                  out_dir = SHOTS / f"{vp_name}-{mode_name}"
                  out_dir.mkdir(parents=True, exist_ok=True)
                  for route in ROUTES["routes"] + ROUTES["drawer_states"]:
                      if ROUTE_FILTER and route["slug"] != ROUTE_FILTER: continue
                      # Skip drawer_states for non-laptop or non-founder per D-05
                      if route in ROUTES["drawer_states"] and (vp_name != "laptop" or mode_name != "founder"): continue
                      ctx_kwargs = {"viewport": {"width":w,"height":h}}
                      if route.get("auth"):
                          ctx_kwargs["storage_state"] = "storage-state.json"
                      ctx = browser.new_context(**ctx_kwargs)
                      ctx.add_init_script(f"""
                          localStorage.setItem('explainMode','{mode_cfg['explainMode']}');
                          localStorage.setItem('devMode','{mode_cfg['devMode']}');
                      """)
                      page = ctx.new_page()
                      url = ROUTES["base_url"] + route["path"]
                      try:
                          page.goto(url, wait_until="domcontentloaded", timeout=30000)
                          if route.get("wait_selector"):
                              page.wait_for_selector(route["wait_selector"], timeout=15000)
                          page.wait_for_timeout(int(route.get("wait_ms", 800)))
                          page.add_style_tag(content=DETERMINISTIC_CSS)
                          page.wait_for_timeout(200)
                          png_path = out_dir / f"{route['slug']}.png"
                          page.screenshot(path=str(png_path), full_page=False)
                          size = png_path.stat().st_size
                          mf.write(f"{route['slug']}\t{vp_name}\t{mode_name}\t{url}\t{png_path}\t{size}\n")
                          shots += 1
                          print(f"[capture] {vp_name}/{mode_name}/{route['slug']} -> {size}B")
                      except Exception as e:
                          print(f"[capture:FAIL] {vp_name}/{mode_name}/{route['slug']}: {e}", file=sys.stderr)
                          fails += 1
                      finally:
                          ctx.close()
      browser.close()
  print(f"[capture] phase={PHASE} shots={shots} fails={fails}")
  sys.exit(1 if fails > 0 else 0)
  PY
  ```
