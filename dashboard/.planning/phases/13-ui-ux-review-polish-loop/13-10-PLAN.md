### Agent card template (per MC reference/agents.png vision)
```tsx
<div className="group relative rounded-lg border border-[color:var(--border)] bg-[color:var(--bg-elev)] p-4 hover:border-[color:var(--accent)]/20 transition-colors">
  <div className="flex items-start justify-between gap-3">
    <div className="flex items-start gap-3">
      <Avatar name={agent.name} size={40} />
      <div>
        <div className="text-[15px] font-semibold">{agent.name}</div>
        <div className="mt-0.5 text-[12px] font-mono text-[color:var(--text-muted)]">{agent.model}</div>
      </div>
    </div>
    <StatusPill active={agent.active} />
  </div>
  <div className="mt-3 flex items-center justify-between text-[12px] text-[color:var(--text-muted)]">
    <span>{agent.lastActive ? `${relative(agent.lastActive)}` : "Never"}</span>
    <span>{agent.successRate7d != null ? `${Math.round(agent.successRate7d * 100)}%` : "—"}</span>
  </div>
  <div className="mt-4 flex gap-2 opacity-0 group-hover:opacity-100 transition-opacity">
    <button className="flex-1 rounded border border-[color:var(--border)] px-2 py-1 text-[12px] hover:bg-[color:var(--bg)]">{verbs.primary}</button>
    <button className="flex-1 rounded border border-[color:var(--border)] px-2 py-1 text-[12px] hover:bg-[color:var(--bg)]">{verbs.stop}</button>
    <button className="flex-1 rounded border border-[color:var(--border)] px-2 py-1 text-[12px] hover:bg-[color:var(--bg)]">{verbs.archive}</button>
  </div>
</div>
```
Hover-reveal verbs is a deliberate choice: reduces visual noise at rest (MC's pattern), keeps affordance discoverable. For keyboard users, focus-within also reveals (`group-focus-within:opacity-100`). Ensure 24×24 click targets per WCAG.
