- Filter pills: level (debug/info/warn/error), scope (chat/api/agent/etc), source (file path)

- Collapsible JSON: detect JSON values, render fold/expand triangles
- Live tail toggle: SSE auto-scroll, pause on user-scroll-up, resume on scroll-to-bottom

**Tests:** virtualization at 10k lines, filter pills update view, copy-as-cURL composes correctly, search highlight works.

## 5.2 — Notifications drawer (E12)

**Component:** `components/shell/notifications-drawer.tsx`
**Trigger:** Bell icon in top-nav (new, with red dot when unread > 0)

**Storage:** `.cae/notifications.jsonl` append-only. Generated by:
- post-commit hook (commits to current branch)
- circuit-breakers events (forge_begin/forge_end)
- workflow-state events (run started/completed/failed)
- token-budget events (crossed 50% / 80% / 100%)
- skill installs / updates
- system errors (from existing IncidentStream)

**UI:**
- Drawer (right-side, 400px) with grouped notifications by source (Build / Workflow / Cost / System)
- Each notification: icon + title + relative time + click → drill to source
- Mark-all-read button
- Filter: unread / all / per-source
- Keyboard: ⌘B opens drawer

**Test:** seed notifications, verify grouping + click drills + mark-as-read state.

**Commit:** `feat(notifications): drawer + bell + per-source groups + auto-events`

## 5.3 — Since-you-left card (E13)

**Component:** `components/build-home/since-you-left.tsx`
**Mount:** /build/page.tsx — visible only when last visit > 1h, dismissible

**Data:** diff `.cae/sessions/last-seen.json` against current state. Show:
- N phases shipped since (link to flame timeline)
- N commits (with stat: +X lines / -Y lines)
- $X tokens spent (link to /metrics Cost tab)
- N errors hit (link to incident stream)
- Top 3 events (most-significant: phase ships, big commits, large costs)

**UX:** card with subtle gradient background, "X to dismiss" button, auto-dismisses after 24h.

**Storage:** track session via signed cookie + last-seen.json file. Update on every page load.

**Test:** seed last-seen 2h ago, render shows summary; click dismiss removes; reload after dismiss does NOT re-show.

**Commit:** `feat(since-you-left): returning-founder summary card with diff stats`
